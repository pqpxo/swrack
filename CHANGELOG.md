# SWrack v26 — Source Snapshot Notes

## Current application identity

- Product name: **SWrack**
- Header logo, favicon, Apple touch icon, and web manifest included
- Internal Docker service names remain backward-compatible with the original
  Cabinet Canvas deployment

## Main features included

- Interactive rack elevation and equipment placement
- Full, half, quarter, and 0.5U custom equipment placement
- Custom equipment templates with themed icons and custom PNG icons
- Built-in 1U–4U blank panels
- Equipment details, IP addresses, and HTTP links
- Manual and automatic two-minute ICMP ping monitoring
- Green / amber / red rack LED state indicators
- Multiple category selection with muted non-matching equipment
- Cabinet creation and deletion
- Custom template deletion and installed-device removal
- Firefox-safe in-app management dialogs
- Docker persistence for PostgreSQL data and uploaded assets
